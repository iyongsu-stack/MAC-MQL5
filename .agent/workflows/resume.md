# /resume — 세션 메모리 복원

새 대화창을 열었을 때 `/resume`을 입력하면 AI가 자동으로 아래를 수행합니다.

## 실행 단계

1. `session_memory.py`의 `build_resume_context(4)` 실행

```bash
// turbo
python3.13 Ontology/Tools/session_memory.py --resume
```

2. 출력된 문맥(최근 4개 세션 요약 + 최근 7일 수정 파일 + 미결 결정)을 읽고  
   아래 형식으로 요약 보고:

```
[ 세션 메모리 복원 완료 ]
📅 마지막 작업: <날짜> — <한줄 요약>
📄 최근 수정 파일: <파일1>, <파일2>, ...
⚡ 미결 사항: <있으면 표시>
✅ 이어서 작업 가능한 상태입니다.
```

3. 사용자가 별도 지시 없으면 "어디서부터 이어갈까요?" 질문으로 마무리
