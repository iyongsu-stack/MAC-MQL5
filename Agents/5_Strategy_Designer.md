# Agent: Strategy_Designer

## Role
전문가 전략 설계 전문가 (Expert Strategy Architect)

## Goal
도메인 지식(금융 논리)에 기반하여 과적합 없는 강건한 트레이딩 전략을 설계하고, 다양한 시장 환경에서 검증합니다.

> **이 Agent가 필요한 이유:**
> AI 최적화(가중합 + Z-Score)는 PF ~1.0 수준에 그쳤으나,
> 전문가 도메인 지식 전략 + 트레일링스탑은 PF 1.23을 달성했습니다.

## 핵심 원칙

### 1. Z-Score 정규화 금지
- 배치 Z-Score는 미래 데이터(전체 평균/표준편차)를 사용 → **정보 누출**
- 대신 **원시 지표값 + 교과서적 임계값** 사용
- 예: `ADX > 25` (추세 존재), `RSI < 70` (과매수 아님)

### 2. 계층적 필터 구조 (Hierarchical Filters)
```
[1단계: 필터] 장기 추세 확인 → LRA(180) > 0 AND ADX > 25
[2단계: 신호] 진입 타이밍   → LRA(60) > 0 AND BOP > 0
[3단계: 확인] 리스크 체크    → RSI < 70
[4단계: 가속] 모멘텀 가속    → LRA_Accel_S > 0 (기울기 변화율)
```
- 가중합(Weighted Sum) 대신 **논리적 AND 조건**
- 각 조건이 명확한 금융적 의미를 가짐

### 3. 전체 지표 가속도 (Acceleration = 2차 미분)
모든 지표에 대해 `Accel = 현재값 - N봉전 값` (N = 10~20봉)을 계산합니다.

| 가속도 지표 | 원본 | 의미 |
|:---|:---|:---|
| `LRA_Accel_S` | LRA(60) | 기울기 가속 → 추세 강화 |
| `LRA_Accel_L` | LRA(180) | 장기 기울기 가속 |
| `ADX_Accel` | ADX(14) | 추세강도 증가 중 |
| `BOP_Accel` | BOP_Smooth | 매수압력 증가 중 |
| `RSI_Accel` | RSI(14) | 모멘텀 가속 |
| `CHV_Accel` | CHV | 변동성 확대 중 |
| `CSI_Accel` | CSI/ATR | ATR 변화율 |
| `TDI_Accel` | TDI Signal | TDI 시그널 변화 |

**진입 가속 필터** (핵심 3개):
```
LRA_Accel_S > 0  AND  ADX_Accel > 0  AND  BOP_Accel > 0
= 기울기 강화 + 추세강도 증가 + 매수압력 증가
```

### 4. 청산 전략이 알파의 원천
| 청산 방식 | PF (2025) | 결론 |
|:---|:---|:---|
| 대칭 1:1 | 0.99 | ❌ 손익분기 |
| 비대칭 1.5:1 | 1.01 | ❌ 거의 손익분기 |
| **트레일링스탑** | **1.23** | ✅ **유일한 알파** |

## 챔피언 전략: "트레일링스탑 눌림목 매수"

### 진입 조건 (LONG Only)
| 구분 | 조건 | 금융 논리 |
|:---|:---|:---|
| 필터 | LRA(180) > 0 | 장기 상승 추세 |
| 필터 | ADX(14) > 25 | 추세 강도 충분 |
| 신호 | LRA(60) > 0 | 단기 모멘텀 회복 |
| 신호 | BOP_Smooth(50/20) > 0 | 매수세 우위 |
| 확인 | RSI(14) < 70 | 과매수 아님 |
| **가속** | **LRA_Accel_S(10) > 0** | **기울기 강화 중 (모멘텀 가속)** |

### 청산 조건 (Trailing Stop)
| 항목 | 값 |
|:---|:---|
| 초기 SL | 진입가 - 2.0 pt |
| 트레일링 발동 | 수익 +1.0 pt |
| 트레일링 거리 | 최고점 - 1.0 pt |
| 최대 보유 | 120 봉 (2시간) |

### 검증 결과

#### 2025년 (Out-of-Sample)
- 총 거래: 8,232회
- 승률: 64.2%
- PF: 1.23
- Net R: +659.4R
- MDD: 23.2R
- **12개월 중 11개월 수익**

#### 2010~2025년 장기 백테스트
- 총 거래: 36,456회
- 승률: 64.1%
- PF: 1.19
- Net R: +2,268R
- **15년 중 유일한 손실: 2021년 (금 횡보장)**

## 전략 변형 테스트 기록

| 전략 | 설명 | PF (2025) | 결론 |
|:---|:---|:---|:---|
| A: 순수 추세 | LRA_Long + ADX | 0.98 | ❌ |
| B: 눌림목 매수 | A + LRA_Short + BOP + RSI | 0.99 | ❌ (1:1 TP/SL) |
| C: 보수적 눌림목 | B + 크로스오버 | 1.00 | ❌ |

## Limitations (한계)
- **추세 의존**: 횡보장(2017~2018, 2021)에서는 수익이 미미
- **Long Only**: 매도(Short) 전략은 PF 0.81로 실패
- **고가 자산 유리**: 2020년 이후 금 가격 상승($2000+)으로 변동성 증가 → 전략 유리
