## 14-A. 필터 수식 상세 설명

### A1. ATR 가속도 (ATR Acceleration)

**공식**:
```
ATR_가속도 = ATR14[현재봉 i] / ATR14[봉 i-5]
```

**전제: ATR14 계산 과정 (Wilder's Smoothing)**:
```
Step 1: True Range (TR)
  TR[i] = MAX(
      High[i] - Low[i],                      ← 현재 봉의 범위
      ABS(High[i] - Close[i-1]),              ← 갭 상승 반영
      ABS(Low[i]  - Close[i-1])               ← 갭 하락 반영
  )

Step 2: ATR14 초기화 및 갱신
  ATR14[14번째봉] = MEAN(TR[1] ~ TR[14])           ← 최초 14봉 단순평균
  ATR14[i]        = (ATR14[i-1] × 13 + TR[i]) / 14  ← 이후 지수평활

Step 3: 가속도
  atr_accel_5 = ATR14[i] / ATR14[i-5]
```

**값 해석**:

| atr_accel_5 값 | 의미 | 거래 판단 | 검증 승률 |
|:---|:---|:---|:---:|
| < 0.90 | ATR 10%+ 감소 → 변동성 급수축 | ✅ 안전 | 52.8% |
| 0.90 ~ 1.00 | ATR 안정 / 소폭 감소 | ✅ 좋음 | 54.5% |
| 1.00 ~ 1.05 | ATR 소폭 증가 | ⚠️ 보통 | 44.8% |
| 1.05 ~ 1.10 | ATR 증가 중 → 변동성 팽창 | ❌ 위험 | 31.0% |
| > 1.50 | ATR 폭등 → 스파이크 진행 중 | ❌ 최악 | 21.7% |

**수치 예시**:
```
[안전 예시]
  봉 i-5: ATR14 = 0.30
  봉 i:   ATR14 = 0.27
  atr_accel_5 = 0.27 / 0.30 = 0.90  → "변동성 감소 중" ✅ 진입 가능

[위험 예시]
  봉 i-5: ATR14 = 0.25
  봉 i:   ATR14 = 0.40
  atr_accel_5 = 0.40 / 0.25 = 1.60  → "변동성 폭등 중" ❌ 진입 금지
```

**필터 조건**: `atr_accel_5 < 1.0` (ATR이 5봉 전보다 줄어들고 있을 때만 진입)

### A2. 눌림목 (Pullback Depth)

**공식**:
```
recent_high_30 = MAX(High[i-29], High[i-28], ..., High[i])   ← 직전 30봉 중 최고가
pullback_depth = (recent_high_30 - Close[i]) / ATR14[i]
```

**시각화**:
```
                   recent_high_30 (30봉 중 최고점)
                       ▼
가격   ─────────────╱╲
                  ╱    ╲
                ╱       ╲
              ╱          ╲──── Close[현재] ← 여기서 진입
             │             │
             │← 이 거리 →│  = pullback_depth (ATR 단위)
```

**값 해석**:

| pullback_depth | 의미 | 판단 |
|:---|:---|:---|
| = 0 | 현재가 = 최고가 → 신고가 | ❌ 천장 위험 |
| 0.1 ~ 0.4 ATR | 고점 바로 근처 | ❌ 위험 |
| **0.5 ~ 2.0 ATR** | **적당히 눌린 자리** | **✅ 진입 최적** |
| > 2.0 ATR | 너무 많이 빠짐 | ❌ 추세 붕괴 가능 |
| > 3.0 ATR | 급락 | ❌ 추세 전환 |

**수치 예시**:
```
[진입 적합]
  직전 30봉 최고가 = 1320.50
  현재 Close       = 1320.20
  ATR14            = 0.30
  pullback_depth   = (1320.50 - 1320.20) / 0.30 = 1.0  → ✅ 적당한 눌림

[진입 부적합 - 과도한 눌림]
  직전 30봉 최고가 = 1320.50
  현재 Close       = 1319.80
  ATR14            = 0.30
  pullback_depth   = (1320.50 - 1319.80) / 0.30 = 2.33 → ❌ 너무 눌림

[진입 부적합 - 고점 근처]
  직전 30봉 최고가 = 1320.50
  현재 Close       = 1320.40
  ATR14            = 0.30
  pullback_depth   = (1320.50 - 1320.40) / 0.30 = 0.33 → ❌ 천장 위험
```

**필터 조건**: `0.5 ≤ pullback_depth ≤ 2.0`

### A3. 모멘텀 (Price Momentum)

**공식**:
```
price_mom_10 = (Close[현재봉 i] - Close[10봉 전 i-10]) / ATR14[i]
```

**값 해석**:

| price_mom_10 | 의미 | 판단 |
|:---|:---|:---|
| **> 0** | 10봉 전보다 현재가 높음 → **상승 추세 유지** | **✅ 진입** |
| = 0 | 10봉간 가격 변화 없음 → 횡보 | ❌ 제외 |
| **< 0** | 10봉 전보다 현재가 낮음 → **하락 전환** | ❌ 제외 |

**수치 예시**:
```
[상승 중]
  10봉 전 Close = 1318.50
  현재 Close    = 1320.20
  ATR14         = 0.30
  price_mom_10  = (1320.20 - 1318.50) / 0.30 = +5.67 → ✅ 상승 중

[하락 중]
  10봉 전 Close = 1322.00
  현재 Close    = 1319.80
  ATR14         = 0.30
  price_mom_10  = (1319.80 - 1322.00) / 0.30 = -7.33 → ❌ 하락 중
```

**필터 조건**: `price_mom_10 > 0`

### A4. 세 필터의 시너지 (동시 충족 시)

```
              30봉 최고점
                ▼
가격   ────╱╲
         ╱    ╲          ← pullback 0.5~2 ATR (고점에서 적당히 눌림)
       ╱       ╲
     ╱    ★진입  ╲       ← 여기서 들어감
   ╱               ╲
  │                  │
  │     10봉 전       │
  │     Close        │
  ★ ← 이 위치가 현재보다 낮아야 함 (mom > 0)
```

**동시 충족 의미**:
1. **ATR 가속도 < 1.0** → "변동성이 줄어들고 있다" = 스파이크 끝, 안정화
2. **눌림목 0.5~2ATR** → "고점에서 적당히 빠졌다" = 천장/바닥 아님
3. **모멘텀 > 0** → "10봉 전보다 높다" = 추세 살아있음

세 조건이 **동시에 만족** = **"상승 추세 중 적당한 눌림에서, 변동성 안정 구간에 진입"**

이것이 **추세 눌림목 매수의 정석**을 수학적으로 구현한 것이며, Close 기준 **70.9% 승률**을 달성.
